Chrome打开或关闭快捷静音
2022-07-17

Chrome打开快捷静音

  1. 安装最新的Chrome
  2. 在Chrome中输入网址: chrome://flags
  3. 在搜索栏中搜索 tab audio muting UI control
  4. 将此选项更改为 Enabled
  5. 重启Chrome

Chrome关闭快捷静音

  1. 安装最新的Chrome
  2. 在Chrome中输入网址: chrome://flags
  3. 在搜索栏中搜索 tab audio muting UI control
  4. 将此选项更改为 Disabled
  5. 重启Chrome
