简介

本文参考 Markdown Preview Enhanced

Markdown Preview Enhanced 是一款为 Atom 以及 Visual Studio Code 编辑器编写的超级强大的 Markdown 插件。 这款插件意在让你拥有飘逸的 Markdown 写作体验。

安装插件

VScode 版本安装教程：
通过VScode安装（推荐）

  1. VScode官网 Visual Studio Code下载并安装VScode。
  2. 打开 vscode 编辑器，在插件页搜索 markdown-preview-enhanced，接着点击 Install 按钮。

HTML 导出

使用
右键点击预览，然后点击 HTML 标签。
接着选择：

  • HTML (offline) 选择这个选项如果你要离线使用这个 html 文件。
  • HTML (cdn hosted) 选择这个选项如果你要远程或在服务器上使用这个 html 文件。

设置
默认：

如果 embed_local_images 被设置为 true，那么所有的本地图片将会被嵌入为 base64 格式。

如果 toc 被设置为 false，那么边栏目录将会被隐藏。如果 toc 被设置为 true，那么边栏目录将会被缺省启动并显示。如果 toc 没有被设置，那么缺省边栏目录将会被启动，但是并不显示。
